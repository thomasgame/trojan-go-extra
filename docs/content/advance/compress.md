### 说明

在客户端与服务器之间对 Trojan 负载做 **zstd** 压缩，可减少跨公网传输的字节量。压缩层位于 Shadowsocks（若启用）与 Trojan 协议之间；**默认开启**，无需写 `compress` 配置。

### 关闭压缩

与未支持该特性的旧版本或其它分支互通时，**客户端与服务端须同时**配置：

```json
"compress": {
    "enabled": false
}
```

### 可选参数

| 字段 | 说明 | 默认 |
|------|------|------|
| `enabled` | 是否启用；未配置视为开启 | 开启 |
| `algorithm` | 算法（当前实现为 zstd） | zstd |
| `level` | 压缩级别 | 3 |
| `min_size` | 小于该字节数不压缩 | 256 |
| `buffer_size` | 压缩块大小（字节） | 32768 |

### 注意

- 两端压缩开关必须一致，否则无法建立连接。
- 当前仅对 TCP 连接生效，UDP 未实现。
